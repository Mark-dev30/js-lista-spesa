PROBLEMA:
Data una lista della spesa, stampare sulla pagina (anche brutalmente, basta che si vedano) gli elementi della lista individualmente con un ciclo while.

RISOLVO:

1 - Creo una lista.
    1.1 - Creo un array con nome "list" acui assegno 1+ stringhe.
    1.2 - Creo una variabile "shopping_list" a cui assegno la classe dell'html.
    1.3 - Creo una variabile "items_list" a cui assegno una stringa vuota.
    1.4 - Creo una variabile "i" a cui assegno il valore 0.
    1.5 - Creo un ciclo while che deve rispetta la condizione : variabile i < della lunghezza della variabile list.
    1.6 - Allinterno del ciclo dichiaro una variabile "item" a cui assegno list[i].
    1.7 - Assegno alla variabile items_list il tag che si deve inserire nella pagina html.
    1.8 - Inserisco items_list all'interno di shopping_list.
    1.9 - Incremento la variabile i di 1.
2 - Visualizzo a schermo la lista.
